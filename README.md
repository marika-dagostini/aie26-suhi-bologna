Questa repository contiene una web-map interattiva sviluppata per visualizzare i risultati presentati nell’abstract:

> **“Isole di calore urbane e trasformazioni territoriali: stima spazio-temporale bayesiana ad alta risoluzione nel comune di Bologna”**

L’abstract è stato presentato al **Congresso dell'Associazione Italiana di Epidemiologia 2026**.
Maggiori informazioni sul congresso sono disponibili al seguente link: [Congresso AIE 2026](https://www.epidemiologia.it/notizie/l-congresso-dell-associazione-italiana-di-epidemiologia-2026)

👉 Per visualizzare la web-map: [https://marika-dagostini.github.io/aie26-suhi-bologna/](https://marika-dagostini.github.io/aie26-suhi-bologna/)
